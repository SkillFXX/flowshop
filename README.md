# Boutique en ligne de vente de produits digitaux

Ce projet est une boutique en ligne dédiée à la vente de produits numériques. Il propose les fonctionnalités classiques d’un site e-commerce : panier, paiement, gestion des comptes clients, connexion via code à usage unique (OTP), génération de factures, envoi d’e-mails, espace de téléchargement et panneau d’administration.

Le projet est basé sur Flask et Jinja côté serveur, avec DaisyUI pour l’interface. Le choix a été fait de rester sur une stack simple, sans framework JavaScript, afin de privilégier les performances et la réactivité.

Le site fonctionne sans AJAX.
